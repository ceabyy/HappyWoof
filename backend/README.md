This is a starter project for me to get familiar with certain libraries i.e. pyTorch, Flask, tkinter.
The reason for this is because I want to understand how to implement my knowledge of AI/ML from my degree practically.
So, a lot of this project in particular is vibe coded. I've taken my time to understand the code and the
differences in choice of models.

I particularly stuck with ResNet18 as ResNet is a model I've studied through for my bachelor's thesis, and
ResNet18 is a quicker and more efficient option than ResNet50, for instance. Additionally, my resources 
with regards to training models are limited, hence this is what I can work with for now.

I learned most about pyTorch and how the models are formed using this library, as I was previously more 
familiar with TensorFlow. I appreciate how much more you can do and have control over.

The model was trained on a dataset from Kaggle - the dogs' emotions can be classified as angry, sad, relaxed or happy.

At the moment, only the model and a python terminal interface is done. In the future, I want to integrate the model
to be interacted with the user through a website, so I will use Flask for this.
